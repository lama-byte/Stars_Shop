# ⭐ Star's Crochet Shop

A simple PHP-based e-commerce website that allows users to browse handmade crochet products, manage a shopping cart, and complete a checkout process.

This project was developed as part of the **Web Programming Laboratory** at **Taibah University**.

---

## ✨ Features

- Beautiful landing page
- User Login & Registration
- Product Catalog
- Shopping Cart
- Add & Remove Products
- Update Cart Quantities
- Checkout Page
- Stock Management
- Category Filtering
- Session-Based Authentication
- Responsive and clean interface

## 🚀 Getting Started

### Prerequisites

- XAMPP
- PHP 8+
- Web Browser

### Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/stars-crochet-shop.git
```

2. Move the project into your XAMPP `htdocs` folder.

```
C:\xampp\htdocs\
```

3. Start **Apache** from the XAMPP Control Panel.

4. Open your browser and navigate to:

```
http://localhost/Star-Shop/
```

## 📌 Main Features

### Product Catalog

- Browse handmade crochet products
- Filter by category
- View product descriptions
- View stock availability

### Shopping Cart

- Add products
- Remove products
- Increase or decrease quantity
- Clear cart
- Automatic subtotal calculation

### Checkout

- Displays complete order summary
- Calculates:
  - Subtotal
  - 15% Tax
  - Total
- Validates stock before completing the order

### Authentication

- User login
- Registration
- Session management
- Logout functionality
